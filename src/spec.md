# Specification

## Summary
**Goal:** Add an admin dashboard to view all demo class booking submissions.

**Planned changes:**
- Create a new admin dashboard page at /admin route that displays all demo class bookings in a table format
- Add backend query method to retrieve all stored demo class booking submissions
- Connect the admin dashboard to the backend using React Query for real-time data fetching
- Add a navigation link in the Header component to access the admin dashboard

**User-visible outcome:** Administrators can navigate to the admin dashboard to view all submitted demo class bookings with details including name, class, course, and submission timestamp in a table format.
